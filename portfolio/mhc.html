Urban policy, Python

Data science-focused urban policy class culminating in a final project on the effectiveness bike lanes in NYC.

About the project

This special section of the Macaulay Honors class "Shaping the Future of New York" explored issues facing the future of New York City using data science and the Python programming language. My final project analyzed and visualized the effectiveness of bike lanes using open source city data, the Python libraries pandas and folium, and GeoJSON.
